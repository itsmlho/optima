# 🎨 OPTIMA Standard Components Reference

> **Berdasarkan**: Pattern terbaik dari `marketing/spk.php` dan `marketing/di.php`
> **Tujuan**: Konsistensi visual dan profesional di seluruh aplikasi

## 📊 Statistics Cards - Standard Pattern

### ✅ Professional Pattern (GUNAKAN INI)
```html
<!-- Statistics Cards - Professional Standard -->
<div class="row g-4 mb-4">
    <div class="col-xl-3 col-md-6">
        <div class="card card-stats bg-primary text-white h-100">
            <div class="card-body d-flex align-items-center">
                <div class="flex-grow-1">
                    <h2 class="fw-bold mb-1" id="stat-value">0</h2>
                    <h6 class="card-title text-uppercase small mb-0">TITLE</h6>
                </div>
                <div class="ms-3">
                    <i class="fas fa-icon fa-2x opacity-75"></i>
                </div>
            </div>
        </div>
    </div>
</div>
```

### 🎨 Color Variants
- `bg-primary` - Biru profesional (#0061f2)
- `bg-success` - Hijau sukses (#00ac69) 
- `bg-warning` - Kuning peringatan (#f4a100)
- `bg-danger` - Merah bahaya (#e81500)
- `bg-info` - Cyan informasi (#00cfd5)
- `bg-secondary` - Abu-abu netral (#6c757d)

### 📏 Key Features
- **Hierarchy**: `h2` untuk nilai, `h6` untuk label
- **Layout**: `d-flex align-items-center` dengan `flex-grow-1`
- **Spacing**: `g-4 mb-4` untuk grid gap dan margin bottom
- **Icons**: `fa-2x opacity-75` untuk ukuran dan transparansi konsisten
- **Responsive**: `col-xl-3 col-md-6` untuk 4 kolom desktop, 2 tablet
- **Height**: `h-100` untuk tinggi konsisten

## 🗂️ Navigation Tabs - Standard Pattern

### ✅ Professional Pattern (GUNAKAN INI)
```html
<ul class="nav nav-tabs mb-3" id="filterTabs">
    <li class="nav-item">

    </li>
    <li class="nav-item">
        <a class="nav-link filter-tab" href="#" data-filter="status">Status Name</a>
    </li>

```

### 📏 Key Features
- **Classes**: `nav nav-tabs mb-3`
- **Active State**: `nav-link active` untuk tab aktif
- **Spacing**: `mb-3` untuk margin bottom konsisten
- **Function**: `filter-tab` class untuk JavaScript handling

- **Spacing**: `mt-3` top margin, `mb-0` bottom

## 🔍 Search & Entries Control - Standard Pattern

### ✅ Professional Pattern (GUNAKAN INI)
```html
<div class="d-flex justify-content-between align-items-center mb-3">
    <div class="d-flex align-items-center gap-2">
        <span>Show</span>
        <select class="form-select form-select-sm" id="entriesPerPage" style="width: auto;">
            <option value="10">10</option>
            <option value="25">25</option>
            <option value="50">50</option>
            <option value="100">100</option>
        </select>
        <span>entries</span>
    </div>
    <div class="d-flex align-items-center gap-2">
        <span>Search:</span>
        <input type="text" class="form-control form-control-sm" id="search" placeholder="" style="width: 200px;">
    </div>
</div>
```

### 📏 Key Features
- **Layout**: Space between untuk entries control (kiri) dan search (kanan)
- **Size**: `form-select-sm` dan `form-control-sm` untuk compact
- **Width**: Auto width untuk select, 200px untuk search input
- **Gap**: `gap-2` untuk spacing antar elemen

## 🏷️ Badges - Standard Pattern

### ✅ Professional Pattern (GUNAKAN INI)
```html
<span class="badge bg-success">Active</span>
<span class="badge bg-warning">Pending</span>
<span class="badge bg-danger">Cancelled</span>
<span class="badge bg-info">Processing</span>
<span class="badge bg-secondary">Draft</span>
```

### 🎨 Status Color Mapping
- **Success** (`bg-success`): Completed, Active, Approved, Ready
- **Warning** (`bg-warning`): Pending, In Progress, Waiting
- **Danger** (`bg-danger`): Cancelled, Rejected, Error, Failed
- **Info** (`bg-info`): Processing, Submitted, Under Review
- **Secondary** (`bg-secondary`): Draft, Inactive, Unknown
